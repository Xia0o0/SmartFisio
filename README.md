# SmartFisio
**SmartFisio** é uma aplicação web responsiva voltada à saúde física na terceira idade, combatendo o sedentarismo e a perda de mobilidade funcional.

## O Projeto
A plataforma centraliza guias de exercícios de baixo impacto, como alongamentos e fortalecimento muscular leve. O grande diferencial é a **acessibilidade**: a interface utiliza elementos de alto contraste, tipografia escalonada e navegação simplificada, pensando em usuários com limitações visuais ou motoras finas.

## Funcionalidades Principais
* **Guias de Exercícios:** Conteúdo especializado para baixo impacto.
* **Monitoramento por Visão Computacional:** (Em desenvolvimento) Uso de IA para auxiliar na execução correta dos movimentos.
* **Acompanhamento de Progresso:** Módulo de persistência de dados para visualizar a evolução de usuário.
* **Interface Inclusiva:** Design focado em UX para idosos (UI de alto contraste e navegação sem complexidade).

## Tecnologias Utilizadas
- **Front-end:** React.js, tailwind CSS/Bootstrap.
- **Back-end:** FastAPI (Python)
- **Inteligência Artificial:** MediaPipe & OpenCV (processamento de imagem).
- **Banco de Dados:** PostgreSQL.
